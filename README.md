<h1>RUSSIAN VERSION</h1>
<p><b>"Вселенная информатики"</b> - это индивидуальный open source проект, связанный с переделкой известного приложения "Мир информатики". Проект реализован на c# с помощью WinForms.</p><br>

<h1>ENGLISH VERSION</h1>
<p><b>"Universe of Informatics"</b> is an individual open source project related to the redesign of the well-known application "World of Informatics". The project is implemented in c# using WinForms.</p>
